# 학생 구강검진 현황판

하루 동안 여러 담임이 동시에 접속해서 학급별 구강검진 상태를 표시하는 간단한 웹앱입니다.

## 실행

```bash
/Users/hazel/.cache/codex-runtimes/codex-primary-runtime/dependencies/node/bin/node server.js
```

실행 후 접속 주소:

- 이 컴퓨터에서: `http://localhost:3100`
- 같은 와이파이에서: `http://이-맥의-IP주소:3100`

## 사용 순서

1. `학급 설정` 버튼을 눌러 학교 이름, 제목, 학급 배치를 수정합니다.
2. 각 학급의 `건물`, `층`, `표시 이름`을 실제 배치도에 맞게 바꾼 뒤 `배치 저장`을 누릅니다.
3. 담임은 자기 기기에서 `내 이름`을 입력합니다.
4. 학급 카드에서 `검진 전`, `검진 중`, `완료` 중 하나를 누릅니다.

## 참고

- `검진 중`은 한 번에 한 학급만 유지되도록 되어 있습니다.
- `상태 초기화`를 누르면 전체 학급 상태와 최근 기록이 초기화됩니다.
- 실제 학교 배치도 파일이 생기면 그 구조에 맞게 본관/신관/별관 층 배치를 더 정확하게 다듬을 수 있습니다.
